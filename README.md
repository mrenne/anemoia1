# anemoia1
An ambient piece made with code.
Software used: Sonic Pi and Ableton Live 12.

Code is being manipulated in Sonic Pi and sent to Ableton Live 12 using the following VSTs:
Drone Piano
Decent Sampler: Venus Theory - Auras: Eraphine
Decent Sampler: Distance Zero DS
Spitfire Audio: Tape Synths by Blankfor.ms
